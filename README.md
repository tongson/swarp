# USAGE
Configured through environment variables

`SWARP_BIND`: IP to bind to, default: 127.0.0.1 <br/>
`SWARP_PORT`: Port to listen from, default: 3030 <br/>
`SWARP_CWD`: Directory to serve, default: . <br/>
